\doxysubsection{Detailed Description}
Returns an Image object that can then be painted on the screen. The url argument must specify an absolute \href{\#{@link}}{\texttt{ @javalink URL\}}}. The name argument is a specifier that is relative to the url argument. 

This method always returns immediately, whether or not the image exists. When this applet attempts to draw the image on the screen, the data will be loaded. The graphics primitives that draw the image will incrementally paint on the screen.


\begin{DoxyParams}{Parameters}
{\em url} & an absolute URL giving the base location of the image \\
\hline
{\em name} & the location of the image, relative to the url argument \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
the image at the specified URL
